# Troubleshooting

If something is not working as expected, check the table below for common problems and solutions.

| Problem | Solution |
|---------|----------|
| The counter is not going up when I tap | Make sure you are tapping the round **Increment** button (the plus icon) in the bottom-right corner of the screen. |
| The counter jumped back to zero | The counter resets when you fully close and restart the app. This is expected behaviour — the count is not saved between sessions. |
| The screen looks blank | Close the app completely and open it again. |
| The app is slow to respond to taps | Try closing other apps running in the background, then reopen this app. |
| I can't find a feature | Check the [manual index](index.md) for a full list of what the app can do. |
